# deep-research

A **Deep Research (DR)** document is an in-depth, exhaustive investigation of a topic. Each DR condenses hundreds of sources into a single, navigable Markdown document with tables, Mermaid diagrams, comparison matrices, and actionable findings.

DRs are **read-only knowledge artifacts**. They are not decision records or project specs — they are the research that informs decisions.
